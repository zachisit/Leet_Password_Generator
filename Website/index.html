Passwordington is the result of a side project by Zach Smith - @zachisit. Site is hosted via an AWS s3 bucket, powered by two AWS Lambda FAAS methods connected to AWS API Gateway endpoints: an AWS Lambda FAAS to generate passwords written in Python and an AWS Lambda FAAS to generate audio file written in Python leveraging AWS Polly service, and utilizing Integer as a Service CountAPI. Fun times. This site has been beta for days, at version 1.2.3.
